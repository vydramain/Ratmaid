# Список доработок

## UX / Интерфейс

- [x] Меню настроек — громкость музыки и звуков отдельно, управление, разрешение
- [x] Меню паузы во время игры (вызов по Escape, кнопки: продолжить / настройки / выйти в главное меню)

> Из фидбэка: игроки отмечали отсутствие возможности настроить звук; несколько человек жаловались на неудобство возврата в меню после смерти.

## Аудио

- [ ] Переписать музыку: отдельные треки для каждой локации
- [ ] Сохранить принцип динамической музыки — тихая фаза при уборке, активная при бою

> Из фидбэка: динамическая музыка (amen break при стрельбе) была отмечена как одна из лучших деталей — стоит развить, а не убирать.

## Графика / Арт

- [ ] Переработать спрайты персонажей, окружения и объектов под перспективу вида снизу (bottom-up): тела, мебель, стены должны выглядеть так, будто смотришь снизу вверх, а не сверху вниз
- [ ] Согласовать кровь и пятна с новой перспективой — они уже на полу, логика правильная, нужна визуальная доработка

> Из фидбэка: несколько игроков не считывали перспективу вообще; YurisCat отметил, что тела рендерятся под кровью — это технически верно для bottom-up, но визуально неочевидно без правильных спрайтов.

## Геймплей / Уборка

- [ ] Мешки для трупов: горничная сначала идёт к источнику мешков, берёт мешок, и только после этого может «упаковать» труп. До упаковки труп нельзя нести — только таскать на месте/откатывать (если такая логика появится). После упаковки горничная носит труп уже в мешке (один мешок = один труп).
- [ ] Уборочная тележка горничной — стандартный инвентарь горничной, который заменит/расширит текущий мусорный бак. На тележке: ведро с водой (для отмывания швабры от крови), стопка мешков для трупов, отделение для упакованных трупов. Тележку можно толкать по уровню; швабра «макается» в ведро, мешки берутся со стопки, упакованные трупы складываются в отделение.
- [ ] Уничтожение трупов в мешках — финальный шаг утилизации, чтобы тело уже нельзя было найти полицией. Возможные варианты: мусоросжигатель/печь на кухне, кислотный бак, промышленная мясорубка, унитаз/канализация (для расчленённых частей), или вынос в фургон-катафалк за пределами карты. На уровне должна быть одна точка-«уничтожитель»; туда горничная подвозит тележку или приносит мешок вручную, после чего труп безвозвратно удаляется (и больше не попадает в условие «грязно/чисто»).
- [ ] Кровавые следы от горничной — когда горничная проходит по луже крови, её ботинки пачкаются и какое-то время (или некоторое расстояние) после этого она оставляет за собой кровавые следы шагов. Следы — это новый «мусор», который тоже надо отмывать шваброй; они учитываются в условии чистоты уровня. Когда кровь на ботинках «кончается» (несколько шагов / выход на чистый пол), следы перестают появляться. Бонус: швабра тоже может оставлять размазанный кровавый след, если ей не «макали» в воду.
